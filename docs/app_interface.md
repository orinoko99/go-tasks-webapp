(мобильная версия — stack: доска сверху, всё остальное под ней с табуляцией)

- **Основной экран** (всегда видна доска):
  - Левая часть (70–75%): интерактивная доска + панель управления под ней
  - Правая часть (25–30%): 
    - Выбор сборника (dropdown или карточки с обложками)
    - Список задач текущего сборника (с иконками прогресса: ✓ решено, • начато, пусто — не тронуто)
    - Вертикальное дерево вариаций (move tree) — показывает основные и побочные линии

- **Элементы управления под доской** (горизонтальная панель):
  - Начать заново / Сброс
  - Подсказка (показать следующий правильный ход мигающим камнем)
  - Показать решение (пройти всю ветку автоматически)
  - Следующая задача / Предыдущая

- **Адаптивность**:
  - На экранах < 900 px → правая панель становится вкладками или выдвижной (drawer)
  - На мобильных — доска на весь экран, панель действий и список задач внизу или в модальных окнах
